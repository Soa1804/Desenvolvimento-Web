## 🎮 Projeto em Destaque

### 🕹️ Site de Jogos

Um site pessoal contendo:

- Página inicial com tabela de jogos
- Página individual para cada jogo
- Fundo personalizado com a capa do jogo
- Informações como:
  - 📅 Data de lançamento
  - 🏢 Desenvolvedora
  - 📖 Sinopse
- 🖼️ Galeria com imagens do jogo
- Layout com tema gamer
